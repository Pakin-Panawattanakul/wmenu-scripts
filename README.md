# wmenu-scripts
Simple wmenu application launcher, powermenu, and screenshot scripts written in POSIX sh

## Credits
- [dmenu_drun](https://github.com/xlucn/dmenu_drun) - original Python implementation this was inspired by

## Scripts
- wmenu-drun - application launcher
- wmenu-power - power menu (mainly for i3 and sway)
- wmenu-screenshot - screenshot tool

## Dependencies
- wmenu
- grim, slurp, wl-copy (screenshot)
- swaylock or i3lock (power menu)
